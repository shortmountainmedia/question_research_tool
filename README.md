# Question Research Tool

A lightweight web app for generating audience question ideas around a topic. It is designed to help with article planning, FAQ drafting, and research briefs inspired by Reddit, Quora, social, and search-engine question patterns.

## How to use

1. Open `index.html` in a browser.
2. Enter a topic such as "AI for small business".
3. Click "Generate questions" to create question prompts.
4. Copy the results for article ideas or SEO research.

## Notes

This first version uses local templates to simulate public search-question patterns. If you want, the next step can connect to real search APIs or scraping services for live question discovery.
